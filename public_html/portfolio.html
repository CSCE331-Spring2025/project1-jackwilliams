single-cycle Arm CPU

For a class project, I built a single-cycle ARMv8 processor in Verilog, implementing a main control unit and integrating components like ALU, register file, and memory to support ARMv8 assembly instructions (LDUR, STUR, ADD, B, etc.). Simulated with test programs to ensure correct operation of control signals and instruction execution. I found it to be the most interesting class project that I have done in school, as I learned about things that I had wondered for years.
